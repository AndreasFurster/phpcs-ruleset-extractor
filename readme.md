# PHPCS Ruleset extractor
## Usage
```
$ git clone https://github.com/AndreasFurster/phpcs-ruleset-extractor
$ cd phpcs-ruleset-extractor
$ git clone https://github.com/squizlabs/PHP_CodeSniffer

// Normal list
$ node extract.js

// Tree structure 
$ node extract.js tree
```
